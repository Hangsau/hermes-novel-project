# 五子棋規則演化參考

## 規則難易度與公平性演化

1. **普通無禁規則 (First Player Advantage)**
   - 特點：先手必勝（經數學證明）
   - 遊戲性：純粹的先手優勢，缺乏策略深度
   - 小說對應：異世界初階地區，主角因現代知識輕鬆稱霸

2. **連珠規則 / Renju (禁手規則)**
   - 特點：添加禁手（雙三、雙四、長連）制衡先手
   - 遊戲性：提升策略性，但先手仍具優勢
   - 小說對應：中階區域，遇到「禁手牢籠」的守護者

3. **塔拉山口-10 (現代主流規則)**
   - 特點：開局階段黑方提供10個非對稱點，白方選擇其一
   - 遊戲性：高度依賴心理戰與預判，被視為目前最公平規則
   - 小說對應：頂點試煉場，最終Boss戰的規則

4. **Swap2 (交換二)**
   - 特點：開局黑白各落二子後，白方可選擇持色或再加一黑一白
   - 遊戲性：徹底消除先手優勢，純粹技術比拼
   - 小說對應：傳說中的「絕對公平試煉」，只有真正的棋才能通關

## 在小說中的應用概念

- 雞的「武力」代表：破壞規則限制、強行連線
- 主角的「五子棋」代表：利用規則漏洞、規則重構
- 故事主題：從「先手必勝」的傲慢，到理解「規則公平」的智慧
