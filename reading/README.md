# 讀書總目錄

## 目錄結構

- **by_book/** → 按書名存放每本書的章節與筆記，空間與進度可獎立即看
- **by_language/** → 按語言/文體分類（古文/現代/日文/輕小說）
- **by_topic/** → 按主題分類（經典/歷史/哲學/奇幻/情愛等）
- **by_date/** → 按年月日期存放當日閱讀日誌與綜合進度

## 使用 Skill

呼叫 `novel-reading` 即可一鍵抓取、切章、排程寫心得。
